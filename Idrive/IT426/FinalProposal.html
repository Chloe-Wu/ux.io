About Course Work

Proposal


The student project management web tool would help the project manager to create the Work Breakdown Structure (WBS) for the project. It can let both the project manager and the team member knows their tasks, predecessor and deadline clearly.

Theme: Student Project Management Web Tool

Target Audience: students who is in project management major at CWU

Sample Used: Time & Inventory & Pie Chart & Progress Report

Data Entry Validation: Task Name, Resource Name (students' name), duration

Graph Use: Slideshow (decoration), pie chart (record who use the most time on the task/project or which task costs the most time in the whole project process)
